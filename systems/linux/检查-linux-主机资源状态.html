检查 Linux 主机资源状态

发表于 2024-01-19 更新于 2024-03-18 分类于 systems
本文字数： 106 阅读时长 ≈ 1 分钟

基本情况

功能 命令
CPU htop
GPU nvidia-smi
磁盘 iostat
端口 sudo netstat -pnltu

相比于 netstat，现在更推荐使用 ss 命令。

网络实时活动

功能 命令
不同连接的网速 sudo iftop
具体接口的不同连接的网速 sudo tcptrack -i eno1
具体端口的网络活动 tcpdump -i any port 53210

注意 docker 容器内的端口号和容器外的端口号是不同的。
