取消 Windows 的 PATH

sudoedit /etc/wsl.conf

这样 PATH 就不会有 Windows 的了，而且 manpath 的输出结果也没有 Windows 目录。这样做 man 速度显著提升。至于 VS Code 等软件的命令，可以单独加入 PATH。
