如何防止别人盗用自己的 LAN 代理？

选择 -1：设置 inbound

https://github.com/Dreamacro/clash/pull/2818

Mixin 算作是配置的补充，发生在 Clash Core 启动完成之后。inbound 必须在 Clash Core 启动时添加。打开 Home Directory，然后打开 config.yaml，修改 inbound 属性。参考：

选择 0：bind *，但在 Windows 中设置防火墙

看起来很复杂。

选择 1：填写 bind 地址（无法解决问题）

点击 Allow LAN 右边的按钮，可以看到本机上常用的网卡。这里可以把 WSL 的网关 ip 填上。缺点：由于只能填写一个 bind 地址，所以这样做本机无法代理！ 重启 clash 之后也会发现 Port 变成了随机的；虽然系统代理打开了，但是 clash 拒绝了 Windows 应用的连接。

选择 2：在配置中添加 authentication（可以一定程度解决问题，但是不完全）

Parser 是每次更新订阅完成之后，对订阅得到的内容进行修改的一种机制，需要更新订阅才能生效，而且不同的订阅可以有不同的 parsers。
